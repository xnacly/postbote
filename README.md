# postbote

A minimalist opinionated mutt alternative Mail client inspired by vi, built for the terminal

## Features

- vi style bindings
- single, opinionated configuration
- go template based scripting and commands system
- IMAP, SMTP support via ProtonMail Bridge or any standard server
- most MIME types supported inline
    - images are rendered to the terminal
    - html is converted to text

### Configuration

See [postbote.toml](./postbote.toml).
